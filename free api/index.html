Overview

The Decora API provides a comprehensive interface for managing home decoration products, user accounts, and e-commerce functionality. Our RESTful API follows industry best practices and returns JSON-encoded responses.

Base URL

https://decora-backend.vercel.app/api

Response Format

All API responses follow a consistent JSON format:

{
  "success": true,        // Boolean indicating success/failure
  "message": "Success",   // Human-readable message
  "data": { ... },        // The actual data payload
  "meta": { ... }         // Pagination or metadata (when applicable)
}

Error Handling

Errors return standard HTTP status codes along with a JSON response:

{
  "success": false,
  "message": "Product not found",
  "error": {
    "code": "PRODUCT_404",
    "details": "The requested product ID does not exist"
  }
}

Rate Limiting

API requests are limited to:

  • 100 requests per minute for authenticated users
  • 30 requests per minute for unauthenticated access

Exceeding these limits will return a 429 Too Many Requests response.

Authentication

Most Decora API endpoints require authentication using JWT (JSON Web Tokens). To authenticate:

  1. Register a user account or login with existing credentials
  2. Receive a JWT token in the response
  3. Include this token in the Authorization header of subsequent requests
Authorization: Bearer your.jwt.token.here

Security Note

Always keep your API tokens secure. Never expose them in client-side code or public repositories. Tokens expire after 24 hours, after which you'll need to login again.

Products API

Manage your home decoration product catalog with these endpoints. Products can include furniture, lighting, wall art, and other home decor items.

GET

Get All Products

Retrieve a paginated list of all available products. Results can be filtered by category, price range, and sorted by various fields.

https://decora-backend.vercel.app/api/products/all

Query Parameters

Parameter Type Description
page integer Page number for pagination (default: 1)
limit integer Number of items per page (default: 10, max: 50)
category string Filter by product category (e.g., "Lighting", "Furniture")
minPrice number Minimum price filter
maxPrice number Maximum price filter
sort string Sort field (e.g., "price", "-price" for descending)

POST

Create Product

Add a new product to the catalog. Requires admin privileges.

https://decora-backend.vercel.app/api/products/add-product
Requires authentication with admin role

Request Body

Field Type Required Description
name string Yes Product name (3-100 characters)
price number Yes Product price (positive number)
description string Yes Detailed product description
category string Yes Product category
stock integer No Initial stock quantity (default: 0)
imageUrl string Yes Main product image URL
images array No Additional image URLs
specifications object No Key-value pairs of product specs

{

  "price": 179.99,
  "description": "Rustic industrial-style bookshelf with iron frame and reclaimed wood shelves...",

  "stock": 25,

PUT

Update Product

Update an existing product's information. Requires admin privileges.

https://decora-backend.vercel.app/api/products/update/{id}
Requires authentication with admin role

Path Parameters

Parameter Type Description
id string The ID of the product to update

Request Body

Include only the fields you want to update. All fields are optional.

Field Type Description
name string Updated product name
price number Updated price
description string Updated description
category string Updated category
stock integer Updated stock quantity
imageUrl string Updated main image URL
images array Updated additional images
specifications object Updated specifications

Example Request

{
  "price": 189.99,
  "stock": 30,
  "specifications": {
    "weight": "3.2 lbs",

  }
}

Users API

Manage user accounts, authentication, and authorization for your e-commerce platform.

POST

Register User

Create a new user account with email and password. Returns a JWT token for authentication.

https://decora-backend.vercel.app/api/users/register

Request Body

Field Type Required Description
name string Yes User's full name
email string Yes Valid email address
password string Yes Password (min 6 characters)

Example Request

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "securepassword123"
}

Example Response

{
  "success": true,
  "message": "User registered successfully",
  "data": {
    "user": {
      "_id": "67b9dbfdd1d649814f77c06d",
      "name": "John Doe",
      "email": "john@example.com",
      "role": "user",
      "createdAt": "2023-06-18T09:15:00.000Z"
    },
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
}
POST

Login User

Authenticate a user with email and password. Returns a JWT token for accessing protected endpoints.

https://decora-backend.vercel.app/api/users/login

Request Body

Field Type Required Description
email string Yes Registered email address
password string Yes Account password

Example Request

{
